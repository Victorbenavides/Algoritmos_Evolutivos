# Algoritmos_Evolutivos
Inspiración en la naturaleza: El código usa la librería inspyred para simular un proceso de evolución. Crea una "población" de posibles respuestas y, mediante selección y mutación, busca que solo las más aptas sobrevivan.

El reto matemático: Se está intentando resolver un problema llamado Schwefel. Es como un mapa lleno de trampas y montañas donde el algoritmo tiene que encontrar el punto exacto más bajo o más alto sin perderse en el camino.

Configuración del sistema: Se definieron 100 individuos que van evolucionando a lo largo de 30,000 intentos. Además, se incluyó el "elitismo", lo que garantiza que la mejor solución encontrada nunca se pierda y siempre pase a la siguiente ronda.

El resultado: Al final, la computadora lanza una cadena de ceros y unos que representa la respuesta óptima. Es el "ADN" de la solución que mejor resolvió el problema tras miles de generaciones.
